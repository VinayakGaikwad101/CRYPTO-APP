# CRYPTO-APP
A full stack crypto trading app with react for frontend, and spring boot for backend

# Tech Stack:
## Frontend:
1) React
2) Tailwind
3) Redux
4) ShadCN UI

## Backend and DB:
1) Spring boot
2) Spring security
3) jwt
4) Java mailsender
5) mysql db

## API:
1) Gemini API
2) CoinGecko API
